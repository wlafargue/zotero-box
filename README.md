# zotero-box

## Objective

Enables to set output BibTeX format for Zotero software to the following format:

"""
@article{NameYEAR_first-word-title,\
    title = {Title},\
    volume = {VOL},\
    issn = {ISSN},\
    url = {url},\
    doi = {doi},\
    number = {NUMBER},\
    journal = {Journal},\
    author = {Authors},\
    month = month,\
    year = {YEAR},\
    pages = {PAGES}\
}
"""

## Installation

Copy/paste the BibTeX.js file into $ZOTERO/translators directory
